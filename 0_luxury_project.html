Luxury Wheels | Project

Project:

The Luxury Wheels project was developed as the final project for the Python course, achieving a score of 90%. This project simulates a luxury car rental company and features a backend built with Python, utilizing the Flask framework to integrate with the frontend. The database is managed using SQLite3, containing preloaded data and allowing users to create new records through forms. The design is custom-made, incorporating stylized elements in CSS, along with animations and JavaScript logic throughout various parts of the site. This comprehensive and well-structured project demonstrates a high level of proficiency and attention to detail.

Objectives:

  • check_circleProvide a secure and efficient platform for luxury vehicle reservations.
  • check_circleOffer a user-friendly and intuitive interface for users.
  • check_circleAllow all types of modifications, including changing dates, canceling reservations, and adding extra services to the vehicles.

Technologies Used:

The project was developed using the following technologies:

  • check_circleHTML5, CSS3, JavaScript, Python.
  • check_circleFrameworks: Flask.
  • check_circleDatabase: SQLite3.

Main Features:

  • check_circleUser registration and authentication.
  • check_circleVehicle listing and search.
  • check_circlePassword recovery.
  • check_circleEmail notification upon vehicle reservation, containing all reservation details.
  • check_circleUse of multiple forms to submit data to the back-end and create new database entries.
  • check_circleUtilization of various data types in the database for record creation.
  • check_circleDatabase manipulation, data search, and data download.
